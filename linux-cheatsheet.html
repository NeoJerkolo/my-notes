📁 Навігація та файлова система
$pwd
Показує поточну директорію (Print Working Directory).
ПРИКЛАДpwd
ВИВІД/home/user/projects
$ls
Виводить список файлів та директорій у поточній папці.
ПРИКЛАДls -la
— список з прихованими файлами та правами
-l докладний вигляд -a приховані файли -h розмір у KB/MB
$cd
Перехід між директоріями (Change Directory).
ПРИКЛАДcd /var/log
НАЗАДcd ..
ДОМАШНЯcd ~ або cd
$find
Пошук файлів за іменем, типом або датою зміни.
ПРИКЛАДfind . -name "*.log"
ТИПfind /etc -type f -name "nginx*"
✏️ Робота з файлами
$cp
Копіює файли або директорії.
ФАЙЛcp file.txt /backup/file.txt
ПАПКАcp -r ./src ./dist
-r рекурсивно (папки) -v показувати процес
$mv
Переміщує або перейменовує файли та директорії.
ПЕРЕМІСТИТИmv file.txt /tmp/
ПЕРЕЙМ.mv old.txt new.txt
$rm
Видаляє файли або директорії. ⚠️ Без кошика — назавжди!
ФАЙЛrm file.txt
ПАПКАrm -rf ./node_modules
-r рекурсивно -f без підтвердження -i з підтвердженням
$mkdir
Створює нову директорію.
ПРОСТАmkdir myproject
ВКЛАДЕНІmkdir -p a/b/c/d
$cat / less / head / tail
Читання вмісту файлів різними способами.
catcat file.txt — вивести весь вміст
lessless file.txt — посторінково (q — вихід)
headhead -n 20 file.txt — перші 20 рядків
tailtail -f app.log — слідкувати за логом
$chmod
Змінює права доступу до файлу (read/write/execute).
ВИКОНАВ.chmod +x script.sh
ЦИФРИchmod 755 deploy.sh
755 = rwx r-x r-x (власник / група / інші)
⚙️ Процеси та система
$ps
Показує запущені процеси.
ВСІps aux
ПОШУКps aux | grep nginx
$kill / killall
Завершує процес за його PID або назвою.
ЗА PIDkill 1234
ПРИМУС.kill -9 1234
ЗА ІМ.killall node
$top / htop
Моніторинг процесів у реальному часі. htop — зручніша версія з кольорами.
ЗАПУСКtop
КРАЩИЙhtop
q — вихід, k — вбити процес
$df / du
Перевірка вільного місця на диску та розміру директорій.
ДИСКdf -h
ПАПКАdu -sh ./dist
ТОП-10du -sh * | sort -rh | head -10
🌐 Мережа
$ping
Перевіряє доступність хоста в мережі.
ПРИКЛАДping google.com
N РАЗІВping -c 4 8.8.8.8
$curl
HTTP-запити з командного рядка. Завантаження файлів та тестування API.
GETcurl https://api.example.com/users
POSTcurl -X POST -d '{"name":"test"}' URL
ФАЙЛcurl -O https://example.com/file.zip
$ssh
Підключення до віддаленого сервера через SSH.
БАЗОВОssh user@192.168.1.10
ПОРТssh -p 2222 user@host
КЛЮЧssh -i ~/.ssh/id_rsa user@host
$scp
Копіювання файлів між локальною машиною та сервером через SSH.
НА СЕРВЕРscp file.txt user@host:/tmp/
З СЕРВЕРАscp user@host:/var/log/app.log .
🔍 Пошук та обробка тексту
$grep
Пошук тексту за патерном у файлах або виводі команд.
У ФАЙЛІgrep "ERROR" app.log
РЕКУРС.grep -r "TODO" ./src
РЯДОК №grep -n "function" main.js
-i без регістру -v інвертувати -c кількість
$| (pipe)
Передає вивід однієї команди на вхід іншої. Основа Linux-магії.
ПРИКЛАДps aux | grep node | wc -l
ЛОГИcat app.log | grep ERROR | tail -50
$wc
Підрахунок рядків, слів, символів у файлі.
РЯДКИwc -l file.txt
СЛОВАwc -w file.txt
$awk / sed
Потужна обробка та редагування тексту рядок за рядком.
awkawk '{print $1, $3}' file.txt
sedsed 's/foo/bar/g' file.txt
sed — замінює текст; awk — обробляє колонки
💡 Корисне різне
$sudo
Виконує команду з правами адміністратора (root).
ПРИКЛАДsudo apt update
ROOTsudo -i — увійти як root
$history
Показує список виконаних команд. Швидкий повтор через !! або !n.
СПИСОКhistory | tail -20
ПОШУКhistory | grep git
ПОВТОР!! — остання команда
$alias
Створює коротке ім'я для довгої команди. Зберігати в ~/.bashrc.
СТВОРИТИalias ll='ls -la'
СПИСОКalias — показати всі
$man
Довідка по будь-якій команді. Найкращий спосіб дізнатись про прапори.
ПРИКЛАДman ls
КОРОТКОls --help
q — вихід з довідки
